    Team Page

    Technical Description

    1. HTML Structure and Key Tags

  • The <body> tag Contains all visible content, including the background images, team slogan, and team member details.

  • The layout is divided into meaningful sections using <div> elements. The background_images div contains cloud images that create a visually appealing backdrop, while the trans_box div houses the motivational slogan.

  • Each member's details are wrapped in <a> tags, making their profile images clickable. The <img> tag displays their photos, and an overlay effect reveals their contributions upon interaction, enhancing user

  • Text formatting is applied through <b> tags to highlight names and roles, making the content more readable. The contributions of each team member are neatly structured using <ul> and <li> tags, ensuring a clear and organized presentation. This combination of structural divisions, text styling, and interactive elements creates an intuitive and visually appealing team page.

    • 2. CSS Properties

  • The CSS for the team page is designed to maintain a visually appealing and well-structured layout. The overall page uses a light blue background, ensuring a clean and inviting look. The body section centers content using display: flex while allowing vertical scrolling when necessary. Google Fonts (Poppins) is applied for a modern and professional typography style.

  • Text styling is applied using .gray_text and .black_text, with distinct colors to highlight important phrases in the team's motto. The positioning of these elements is controlled with absolute values to maintain proper alignment.

  • The .team_members section organizes profile cards in a flexbox layout, evenly spacing them. Each member's profile is structured using div elements containing an image and an overlay that reveals additional details upon hovering. The .overlay effect is implemented with a blur effect (backdrop-filter: blur(500px)) and transitions for smooth interaction.

  • The .custom-list ensures that the contributions of each team member are neatly wrapped while allowing flexibility in arrangement. The .overlay is initially set to opacity: 0 and appears with a subtle transition when hovering over a team member's profile, enhancing user interaction.

    • Accessibility

  • Alt text is provided for images, ensuring they are accessible to visually impaired users. The chosen Poppins font and high-contrast color scheme improve readability.

  • The high-contrast text and background improve readability for users with visual impairments. The hover effects on team member images provide visual feedback, enhancing the interactive experience.
